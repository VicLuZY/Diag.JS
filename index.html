Diag.JS / Typed Diagram Engine

Text-to-SVG schematics for electrical, HVAC, network, fire alarm, and lighting control systems.

Author building-system schematics from one concise DSL.

Diag.JS turns short text models into exportable SVG drawings across five building-system disciplines. The tool combines discipline-specific symbol libraries, lane-aware layout, and renderer-specific connection styling so one authoring model can scale from single-line distribution to campus-scale system diagrams.

Documentation

Reference the language, renderer behavior, and source-backed libraries from the same site.

The documentation site covers the DSL grammar, statement syntax, parameter semantics, rendering model, system-specific authoring guidance, generated symbol and media inventories, public API, architecture, and contribution rules. It is built with VitePress and published alongside the demo at /docs/.
