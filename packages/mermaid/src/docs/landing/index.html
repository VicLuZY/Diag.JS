DiagJS

Single-line diagrams (SLD), flowcharts, and automatic layout. A focused fork of Mermaid for electrical device and cable semantics.

View documentation →

About

DiagJS adds the sld diagram type with ATS, GEN, UTIL, CDP nodes and cable definitions, while keeping optional compatibility with flowchart and graph syntax. Layout is handled by dagre (default) or ELK via config.

This project preserves Mermaid attribution and remains under the MIT License. See the docs and the repository for more.

Example: Single-line diagram (SLD)

Building power from utility and generator through an ATS, switchgear, feeder, and panel to loads.

Open the docs to see this diagram rendered on the home page and in the live editor. 

sld LR
UTL1[Utility]{v="480V"}
GEN1[Generator]{v="480V"}
ATS1[Main ATS]{profile="basic", out_breaker=true}
SWG1[Switchgear]{v="480V"}
UTL1 --> ATS1:norm
GEN1 --> ATS1:emer
ATS1:load ==> SWG1
SWG1 -->|Feeder F-12| CND1{type="EMT", len="12m"} --> CAB1{cond="4C", size="3/0"} --> PNL1[Panel L1]{v="480V"}
PNL1 --> CBR_PNL1_1 --> LGT1[Lighting]
PNL1 --> CBR_PNL1_2 --> RCP1[Receptacles]
